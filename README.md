# AI-собеседник для Python-разработчиков

Тренажёр для подготовки к техническому собеседованию на позицию Python-разработчика.  
AI-собеседник на основе **Llama 3.1 (8B)** задаёт вопросы, оценивает ответы и даёт развёрнутую обратную связь.

---

## Возможности

- **3 уровня сложности**: Junior / Middle / Senior
- **4 темы на выбор**: Python, LLM, SQL, Git
- **Гибкое количество вопросов**: 3, 5 или 10
- **Пропуск вопросов**: если не знаете ответ, просто нажмите Enter (оценивается в 0 баллов)
- **Защита от некорректного ввода**: бот не упадёт от букв или пустых ответов
- **Детальная оценка**:
  - Оценка каждого вопроса (0–10)
  - Персональный фидбек по ответу
  - Сильные и слабые стороны
  - Общий вердикт: *готов / почти готов / не готов*
- **Цветовая индикация** 🟢🟡🔴

## Установка и запуск

### 1. Установите Ollama
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### 2. Скачайте модель
```bash
ollama pull llama3.1:8b
```

### 3. Клонируйте репозиторий
```bash
git clone https://github.com/VikaMironova/LLMHelper.git
cd LLMHelper
```

### 4. Установите зависимости
```bash
pip install langchain_ollama langchain_core
```

### 5. Запустите бота
```bash
python algorithm.py
```

### Как пользоваться
1. Выберите уровень сложности (1–3)

2. Выберите тему (1–4)

3. Выберите количество вопросов (1–3)

4. Отвечайте на вопросы

5. Нажмите Enter, чтобы пропустить вопрос

6. Получите разбор и оценку
